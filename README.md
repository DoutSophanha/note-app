
# 🗒️ Note App

A simple and intuitive Note-taking web application built with HTML, CSS, and JavaScript. This app enables users to create, view, and delete notes effortlessly to keep their thoughts organized.

## Technologies Used

- **HTML** – Structure of the application  
- **CSS** – Styling for a clean and responsive design  
- **JavaScript** – Functionality for adding and deleting notes  

## Features

- Add new notes with custom content  
- Delete notes when no longer needed  
- Clean and user-friendly interface  
- Responsive layout for all screen sizes  

## Getting Started

To run the app locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/note-app.git
2. Navigate into the project folder: `cd note-app`
3. Open `index.html` in your preferred browser.
